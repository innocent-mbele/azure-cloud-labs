# Azure Virtual Network Deployment

Built an Azure Virtual Network with a dedicated Resource Group and management subnet.

## Skills

- Microsoft Azure
- Azure Virtual Network (VNet)
- Resource Groups
- Address Spaces
- Subnets

## Deployment

| Resource | Name |
|----------|------|
| Resource Group | rg-cloud-labs-dev-001 |
| Virtual Network | vnet-cloud-labs-dev-001 |
| Address Space | 10.0.0.0/16 |
| Management Subnet | subnet-management (10.0.1.0/24) |
